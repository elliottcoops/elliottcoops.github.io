Matrix based neural network from scratch

Project overview

An implementation of training a neural network using backpropagation from scratch.

Support for any topology of neural network (e.g. (3,32,16)).

Implemented in Python, leveraging NumPy for efficient matrix manipulation.

Tested using MNIST digit dataset.

Training loop:

Runs for specified epochs and computes Z and A for each layer.

Calculates mean squared error loss for training and validation datasets, followed by a backward pass to update weights.

Backpropagation:

Computes gradients of the loss with respect to weights.

Adjusts weights in the opposite direction to minimise the loss.

Results

Using a topology of (64,16,32,16,10) I was able to attain 97% (2 d.p) accuracy
