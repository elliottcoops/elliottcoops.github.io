Optimisation

Why

A common application of optimisation is to minimise a given function - Find the lowest value in a search space

One approach is brute force, however as the search space increases, so do the number of solutions to test exponentially

We use (meta/hyper-) heuristics to attain an (approximate) solution in a reduced amount of time

Problem

Search space: Sin(x) + Cos(2x)

Domain: [X,N]

Implementation

Heuristic: Local search - Greedily improve the solution with no regard to future moves

Metaheuristic: Perturb and then apply Local search - Apply mutation which randomly picks a new location and then improve solution using local search

Observations

The local search heuristic will quickly get stuck in local optima - It is at the best possible solution it can reach

The iterated local search metaheuristic will visit a series local/optimal solutions and seems to perform better on this problem
